# Design

Дизайн-материалы TeamMaker. Структура повторяет архитектуру продукта: каждый вид спорта — отдельная папка со своими референсами и палитрами.

Дизайн-система общая для всего продукта — применяется к `web` и `mobile`.

## Структура

```
design/
├── table-tennis/    # Модуль настольного тенниса
└── utils/           # Вспомогательные скрипты
```

## Текущий статус

Активный модуль — `table-tennis`. Цветовая палитра готова, типографика и компоненты в работе (см. Issue #1, #2).
