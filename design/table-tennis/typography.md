# Типографика — Настольный теннис

Референсы и рекомендации по шрифтам для дизайн-системы TeamMaker.
Приоритет — тёмная тема, мобильное приложение.

---

## Что используют похожие продукты

| Продукт | Шрифт | Характер |
|---|---|---|
| ESPN | DIN / DIN Next | Технический, чёткий, спортивный |
| FIFA Mobile | Miso | Компактный, читаемый в UI |
| NBA App | Trim / Gotham | Жирный для заголовков, чистый для статистики |
| Challonge | Системный sans-serif | Минималистичный, без акцентов |

---

## Рекомендация для TeamMaker

Принцип: **один шрифт для заголовков** (имена игроков, названия турниров) + **один для интерфейса** (счёт, даты, подписи).

### Основной стек

| Роль | Шрифт | Почему |
|---|---|---|
| Заголовки / имена | [Barlow Condensed](https://fonts.google.com/specimen/Barlow+Condensed) | Спортивный конденсат, хорошо на тёмном, бесплатный |
| UI / тело | [Inter](https://fonts.google.com/specimen/Inter) | Лучший UI-шрифт, табличные цифры, широкая поддержка |

### Альтернативы заголовочного шрифта

| Шрифт | Ссылка | Характер |
|---|---|---|
| Oswald | https://fonts.google.com/specimen/Oswald | Конденсат, ближе к классике, нейтральнее Barlow |
| Bebas Neue | https://fonts.google.com/specimen/Bebas+Neue | Только caps, очень «спортивный», для акцентов |
| Rajdhani | https://fonts.google.com/specimen/Rajdhani | Лёгкий технический оттенок |

## Шкала размеров (Mobile-first)

| Уровень | Размер | Вес | Применение |
|---|---|---|---|
| Display | 32px | 700 | Имя игрока / название турнира |
| H1 | 24px | 700 | Заголовок экрана |
| H2 | 18px | 600 | Секция / раунд |
| Body | 14px | 400 | Основной текст |
| Label | 12px | 500 | Подписи, даты, статус |
| Score | 20px | 700 tabular | Счёт матча (табличные цифры) |

Для `Score` — обязательно `font-variant-numeric: tabular-nums`, чтобы цифры не прыгали.

---

## Заметки

- Системные шрифты (SF Pro на iOS, Roboto на Android) — нулевая нагрузка, но теряем единый вид web и mobile.
- Inter поддерживает `tabular-nums` — критично для счётных таблиц.
- Barlow Condensed Semibold (600) хорошо читается на тёмном фоне начиная с 16px.
