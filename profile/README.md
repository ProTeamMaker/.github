# TeamMaker

Платформа для организации спортивных турниров. Старт — настольный теннис,
дальше — модули под другие виды спорта.

Конечный продукт — нативное мобильное приложение. Веб-версия сейчас —
тестовый полигон для проверки продуктовых гипотез перед нативной разработкой.

## Репозитории

- 🌐 [**web**](https://github.com/ProTeamMaker/web) — веб-версия, MVP / тестовый полигон
- 📱 **mobile** — нативное приложение, целевой продукт (план, пока не создан)
- 🎮 [**ue**](https://github.com/ProTeamMaker/ue) — идея 3D-тренажёра на Unreal Engine (на паузе)

См. `CLAUDE.md` в этом репозитории для полного описания продукта.
