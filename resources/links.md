# TeamMaker — Глобальные ресурсы и ссылки

Ссылки на сайты, приложения и материалы, полезные для разработки и дизайна продукта.
Не привязаны к конкретному виду спорта — это общая база знаний.

Для референсов конкретного модуля (цвета, типографика) — смотри `design/<sport>/references.md`.
